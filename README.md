# Cam-Off
Easy Hacking CCTV Cameras
# Install
```
sudo apt-get install python3
apt-get install git
git clone https://github.com/Sn8ow/Cam-Off.git
pip3 install requests
pip3 install colorama
cd Cam-Off
python3 cam-off.py
```
